## Descripción

* Una galería en la que puedes insertar imágenes con descripciones e incluso emojis

## Requisitos

Tener NetBeans instalado

## Uso

* Ingresa en los espacios los datos correspondientes
* Utiliza el boton "Insertar Imagen" para ingresar la imagen a la galería
* Para ingresar emojis, da click en el botón, y después da click derecho para desplegar el menú
* Usa los botones para moverte entre las imágenes
* Los demás botones se explican por sí solos
